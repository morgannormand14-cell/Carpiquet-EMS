# 🏴 Carpiquet EMS Design System

## Brand Statements

- Motto: **Every watt counts.**
- Slogan: **Intelligent energy management for Zendure.**
- Signature: *Designed with ❤️ in Normandy.*
- Engineering statement: **Engineered for reliability. Built for Home Assistant.**
- Golden rule: **Safety before performance. Always.**

## Palette

| Name | Hex | Use |
|---|---|---|
| Carpiquet Black | `#1C1C1C` | Primary background |
| Normandy Gold | `#D4AF37` | Brand accent and headings |
| Normandy Red | `#C8102E` | Critical state |
| EMS Green | `#2E8B57` | Healthy state |
| EMS Orange | `#FF9800` | Warning state |
| Home Assistant Blue | `#03A9F4` | HA-related links and context |
| White | `#FFFFFF` | Primary text on dark backgrounds |

## Logo Rules

- Preserve aspect ratio.
- Do not distort, recolour or crop the flag and lightning mark.
- Keep clear space around the logo.
- Use the full logo for documentation and banners.
- Use a simplified square icon for Home Assistant or HACS when available.

## Typography

Prefer system-safe fonts in Home Assistant.

- Headings: Noto Sans Bold or equivalent
- Body: Noto Sans or equivalent
- Code: JetBrains Mono or equivalent

## Status Colours

- Green: healthy, export reduction achieved
- Blue: neutral or within target range
- Orange: warning or moderate import
- Red: error, unavailable data or critical safety state
- Grey: unknown

## Dashboard Principles

- Native Home Assistant cards first.
- Clear units and labels.
- No colour-only communication; always add text or icons.
- Mobile-first layout.
- Critical status visible without scrolling.
