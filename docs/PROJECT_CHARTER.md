## Mission

Carpiquet EMS is an open-source Energy Management System for Home Assistant and Zendure energy-storage systems.

Its mission is to improve self-consumption, reduce unwanted grid exchange, coordinate multiple batteries and expose every decision transparently.

## Vision

Carpiquet EMS aims to become a trusted open-source EMS for Zendure systems on Home Assistant by combining:

- safety-first engineering;
- transparent algorithms;
- progressive simulation;
- clear diagnostics;
- simple installation;
- maintainable architecture.

## Core Values

### Safety First
No real control feature is enabled before validation in simulation.

### Transparency
Algorithms, limits, assumptions and decisions are documented.

### Reliability
The system must fail safely when data is missing, invalid or stale.

### Simplicity
Configuration and daily use should remain understandable.

### Community
The project welcomes feedback, testing and contributions.
