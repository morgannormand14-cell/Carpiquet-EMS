# RFC-0001 — Normandy

## Status

Accepted

## Summary

Normandy establishes the project identity, documentation system, engineering standards and preparation for a premium Home Assistant experience.

## In Scope

- project charter;
- roadmap;
- architecture;
- EMS specification;
- design system;
- security and contribution policies;
- project history;
- branding assets;
- dashboard requirements.

## Out of Scope

- real battery control;
- predictive optimisation;
- final battery-balancing algorithm.

## Acceptance Criteria

- all Sprint 1 documents are available;
- logo and release visual are stored in `assets`;
- future changes can reference RFC, ADR and EMS rule identifiers;
- current integration remains simulation-only.
