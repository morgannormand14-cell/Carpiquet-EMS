# Carpiquet EMS History

Carpiquet EMS began in 2026 from a real Home Assistant installation in Carpiquet, Normandy.

The initial system combined:

- a Zendure Hyper 2000;
- a Zendure SolarFlow 2400 Pro;
- a Shelly Pro 3EM;
- local MQTT;
- Home Assistant.

The first goal was simple: coordinate two batteries of different capacities while keeping grid exchange close to zero.

The project adopted a simulation-first approach before any real device control.

## Milestones

- **Foundation** — initial Home Assistant integration and simulation.
- **Horizon** — HACS distribution and expanded diagnostics.
- **Normandy** — identity, documentation and engineering foundations.
- **Lightning** — planned secure real control.
- **Equilibrium** — planned intelligent battery balancing.
- **Carpiquet** — first stable release.
